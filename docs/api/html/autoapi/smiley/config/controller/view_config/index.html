ViewConfig

Configuration of view.
class smiley.config.controller.view_config.ViewConfig¶

Configuration of view. Can init ViewConfig and set configuration of view such as update mode, render hints, context menu policy, attribute and drag mode.

Please ensure that the specific configuration is not none.
